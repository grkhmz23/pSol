# pSOL Protocol - Investment Overview

## Executive Summary

pSOL is a comprehensive privacy ecosystem on Solana combining consumer and enterprise privacy solutions through a dual-layer architecture.

**Two Products, One Ecosystem:**
- **pSOL Token**: Privacy token (1:1 with SOL) for mass market
- **Privacy Protocol**: Privacy infrastructure for developers

**Current Status:**
- Privacy protocol deployed on devnet
- Token program deployed on devnet
- Website: psolprotocol.org
- Contact: @unc_gorkh (Telegram)

## Problem

Public blockchains expose all financial data:
1. Users want privacy for everyday transactions
2. Businesses cannot operate with transparent financials
3. Current solutions are niche (mixers) or non-existent (privacy tokens)

## Solution

### Layer 1: Privacy Protocol
- Encrypted on-chain balances
- Zero-knowledge proofs
- Private transfers
- Developer SDK

### Layer 2: pSOL Token
- SPL token in wallet
- Swap SOL ↔ pSOL
- Private amounts
- Mass market product
